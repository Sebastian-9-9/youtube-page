# youtube-page# 📺 YouTube Homepage Clone

A static **YouTube homepage clone** built using **HTML5 and CSS3**.  
This project replicates the layout of YouTube including the header, sidebar, and video grid section.

---

## 🚀 Features

- Responsive Header Section
  - Hamburger Menu
  - YouTube Logo
  - Search Bar
  - Voice Search Button
  - Notification Icon with count
  - Profile Image

- Sidebar Navigation
  - Home
  - Explore
  - Subscriptions
  - Originals
  - YouTube Music
  - Library

- Video Grid Layout
  - Video thumbnail with duration overlay
  - Channel profile picture
  - Video title
  - Channel name
  - Views and upload time

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
